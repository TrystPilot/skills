# Underworld Biographer

A skill for creating rich, detailed underworld narratives, characters, and worlds.

## Overview

This skill helps create immersive underworld-themed content including:
- Detailed character biographies and backstories
- Underworld mythology and lore
- Dark fantasy settings and world-building
- Narrative frameworks for underworld stories
- Character relationships and hierarchies

## Usage

This skill is most effective when you want to:
1. Develop underworld characters with depth and complexity
2. Create cohesive mythological systems for underworld settings
3. Build interconnected narratives within dark fantasy worlds
4. Establish character motivations rooted in underworld lore
5. Design hierarchies and power structures in underworld societies
